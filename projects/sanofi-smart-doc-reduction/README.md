# Smart Doc Reduction — GxP RAG Pipeline
> 🟢 Production — International pharmaceutical group (CAC 40)

On-premise RAG pipeline over ~500 GxP documents (SOPs & WINs). Multilingual embeddings (multilingual-e5-large) + HDBSCAN clustering + Mistral 7B synthesis. 100% local inference, zero data exfiltration.

**Stack:** Python · HuggingFace · HDBSCAN · Mistral 7B (quantized) · FastAPI · Docker
