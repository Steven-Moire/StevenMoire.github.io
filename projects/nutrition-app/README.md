# Nutrition App — Meal Planning & Nutritional Tracking
> 🟡 Prototype — Projet personnel

Flask app for weekly meal planning and nutritional tracking. Built on the CIQUAL database (345 ingredients, French national food composition reference). 64 recipes encoded in JSON.

**Stack:** Python · Flask · CIQUAL JSON
