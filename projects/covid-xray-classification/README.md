# COVID X-Ray Classification — MLOps Pipeline
> 🟢 Complete — École Liora

4-class CNN (COVID, Lung Opacity, Normal, Viral Pneumonia) on 42,330 chest X-rays, with Grad-CAM interpretability. Industrialized into a full MLOps pipeline with 9 containerized services.

**Stack:** PyTorch · Keras · Streamlit · FastAPI · DVC · MLflow · MinIO · Docker · Kubernetes
